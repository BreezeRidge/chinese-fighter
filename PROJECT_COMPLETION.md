## 🎮 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/BreezeRidge/chinese-fighter.git
cd chinese-fighter

# 2. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行游戏
./run.sh
```
