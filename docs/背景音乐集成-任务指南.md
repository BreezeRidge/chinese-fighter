## 📋 需要的音乐

### 1. 菜单背景音乐（menu.ogg）

**用途**: 主菜单、角色选择界面  
**氛围**: 轻松、悠闲、有中国武术氛围  
**节奏**: 中速（90-110 BPM）  
**时长**: 2-3分钟（循环播放）  
**风格**: 电子、摇滚、中国风

**推荐曲目**:
- "Thatched Villagers" (Incompetech)
- "Eastern Thought" (Incompetech)
- "Carefree" (Kevin MacLeod)

### 2. 战斗背景音乐（battle.ogg）

**用途**: 对战场景  
**氛围**: 紧张、激烈、动感  
**节奏**: 快速（120-140 BPM）  
**时长**: 2-3分钟（循环播放）  
**风格**: 激进摇滚、电子、史诗

**推荐曲目**:
- "Dances and Dames" (Incompetech)
- "Pepper Gun" (Incompetech)
- "Mechanolith" (Kevin MacLeod)
- "Volatile Reaction" (Kevin MacLeod)

### 3. 胜利音乐（victory.ogg）

**用途**: KO后胜利画面  
**氛围**: 欢快、胜利、短促  
**节奏**: 任意  
**时长**: 5-10秒  
**风格**: 凯旋音效、号角

**推荐曲目**:
- "Fanfare for Space" (Kevin MacLeod)
- 短促胜利音效（FreePD）

### 步骤1: 下载音乐

```bash
# 创建音乐目录
mkdir -p assets/music

# 下载音乐（示例）
# 从网站下载MP3文件到assets/music/
```

### 步骤2: 转换为OGG格式（推荐）

```bash
# 安装ffmpeg（如果未安装）
# macOS: brew install ffmpeg
# Ubuntu: sudo apt-get install ffmpeg

# 转换MP3为OGG
ffmpeg -i menu.mp3 -c:a libvorbis -q:a 4 assets/music/menu.ogg
ffmpeg -i battle.mp3 -c:a libvorbis -q:a 4 assets/music/battle.ogg
ffmpeg -i victory.mp3 -c:a libvorbis -q:a 4 assets/music/victory.ogg

# -q:a 4 表示质量等级（0-10，4是平衡质量和文件大小）
```

### 步骤3: 代码集成

修改 `src/sound.py`：

```python
def _load_music(self):
    """加载背景音乐"""
    from config import Paths
    
    music_files = {
        "menu": f"{Paths.MUSIC}/menu.ogg",
        "battle": f"{Paths.MUSIC}/battle.ogg",
        "victory": f"{Paths.MUSIC}/victory.ogg",
    }
    
    self.music_tracks = {}
    for name, path in music_files.items():
        if os.path.exists(path):
            self.music_tracks[name] = path
            print(f"✓ 音乐加载: {name}")

def play_music(self, track_name: str, loop: bool = True):
    """播放背景音乐"""
    if track_name in self.music_tracks:
        pygame.mixer.music.load(self.music_tracks[track_name])
        loops = -1 if loop else 0
        pygame.mixer.music.play(loops)
        pygame.mixer.music.set_volume(self.music_volume)
```

修改 `src/config.py`：

```python
class Paths:
    # ... 现有代码 ...
    MUSIC = "assets/music"
```

### 步骤4: 游戏中调用

在 `src/main.py` 中添加音乐切换：

```python
# 菜单场景
sound_manager.play_music("menu", loop=True)

# 战斗开始
sound_manager.play_music("battle", loop=True)

# KO胜利
sound_manager.play_music("victory", loop=False)
```

## 📝 署名示例

如果使用Incompetech音乐，需在游戏中添加积分：

```python
def draw_credits(self, surface):
    """绘制积分/署名"""
    credits = [
        "Music by Kevin MacLeod (incompetech.com)",
        "Licensed under Creative Commons: By Attribution 3.0",
        "http://creativecommons.org/licenses/by/3.0/",
        "",
        "Tracks used:",
        "- Menu: Thatched Villagers",
        "- Battle: Dances and Dames",
    ]
    
    y = 100
    for line in credits:
        text = self.small_font.render(line, True, WHITE)
        surface.blit(text, (50, y))
        y += 30
```
