## 🎉 v0.6.0 新增功能

### 1. 精灵图动画系统 ⭐

**文件**: `sprite_system.py` (267行)

**核心类**:
- `AnimatedSprite`: 动画精灵管理
- `SpriteSheet`: 精灵图加载器
- `Animation`: 动画数据类

**功能**:
- ✅ 6种动画（idle/walk/jump/attack_light/attack_heavy/hit）
- ✅ 帧动画播放（可配置帧率）
- ✅ 循环/非循环动画
- ✅ 水平翻转支持
- ✅ 占位符自动生成（开发阶段）

**占位符设计**:
```
少林僧: 橙红色矩形
峨眉剑客: 蓝色矩形
武当道士: 绿色矩形
默认角色: 灰色矩形
```

### 2. AI对战系统 ⭐

**文件**: `ai_system.py` (315行)

**AI难度**:
- `EASY`: 反应慢(0.5s)，攻击频率30%
- `MEDIUM`: 正常反应(0.3s)，攻击频率50%
- `HARD`: 快速反应(0.15s)，攻击频率70%

**AI性格**:
- `AGGRESSIVE`: 进攻型，70%重攻击
- `DEFENSIVE`: 防守型，80%轻攻击
- `BALANCED`: 平衡型，50/50分配

**AI决策**:
```python
距离 > 200px  → 接近对手
距离 100-200px → 根据性格决定
距离 < 100px  → 攻击或格挡

威胁检测：
- 对手攻击 → 格挡/闪避
- 血量低 → 增加防守
```

**控制**:
- F5: 启用/禁用AI
- F6: 切换难度
- F7: 切换性格
