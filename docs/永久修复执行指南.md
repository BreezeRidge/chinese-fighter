## 🔍 故障排除

### 问题1: pygame安装失败

**错误**: `error: command 'gcc' failed`

**解决**:
```bash
# macOS
brew install sdl2 sdl2_image sdl2_mixer sdl2_ttf
pip3 install pygame

# Ubuntu/Debian
sudo apt-get install python3-pygame

# Windows
pip install pygame --pre
```

### 问题2: 导入pygame失败

**错误**: `ModuleNotFoundError: No module named 'pygame'`

**解决**:
```bash
# 检查Python版本
python3 --version

# 使用正确的pip
python3 -m pip install pygame

# 或指定用户安装
pip3 install --user pygame
```

### 问题3: 生成的精灵图仍然无法加载

**检查**:
```bash
# 1. 验证文件格式
file assets/sprites/shaolin.png

# 2. 检查文件权限
ls -l assets/sprites/

# 3. 测试Pygame加载
python3 -c "
import pygame
pygame.init()
img = pygame.image.load('assets/sprites/shaolin.png')
print(img.get_size(), img.get_flags())
"
```

### 问题4: 颜色没有差异化

**原因**: HSV变换参数可能需要调整

**解决**: 编辑 `tools/regenerate_palettes_pygame.py`，调整色调偏移值:

# 测试Pygame加载

import pygame
pygame.init()

chars = ['default', 'shaolin', 'emei', 'wudang']
for char in chars:
