### 玩家1（红方）
```
移动:   A(左) D(右) W(跳)
攻击:   F(轻) G(重)
防御:   H(格挡)
特殊:   Q(招式1) E(招式2)
```

### 玩家2（蓝方）
```
移动:   ← → ↑
攻击:   J(轻) K(重)
防御:   L(格挡)
特殊:   Q(招式1) E(招式2)
```

```
F1: 少林僧 vs 峨眉剑客
F2: 峨眉剑客 vs 少林僧
F3: 武当道士 vs 少林僧
F4: 武当道士 vs 峨眉剑客
R:  重新开始
ESC: 退出
```

## 🚀 部署说明

### 环境要求
```
Python: 3.10+
操作系统: macOS/Windows/Linux
依赖: pygame-ce, numpy
```

### 安装步骤
```bash
# 1. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行游戏
./run.sh
# 或
cd src && python main.py
```

### 打包发布（可选）
```bash
# 使用 PyInstaller 打包成可执行文件
pip install pyinstaller
pyinstaller --onefile --windowed src/main.py
```

## 📞 技术支持

### 常见问题
**Q: 如何切换角色？**  
A: 按F1-F4快捷键

**Q: 特殊招式如何使用？**  
A: Q/E键，注意冷却时间

**Q: 没有声音？**  
A: 检查系统音量，游戏会自动生成音效

**Q: 如何调整音量？**  
A: 修改 `src/sound.py` 中的 `sfx_volume` 和 `music_volume`
