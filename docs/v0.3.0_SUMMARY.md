## 🎯 设计理念总结

### 代码设计
1. **数据驱动**: 添加角色/招式只需配置，无需修改核心逻辑
2. **模块化**: 7个独立模块，职责清晰，易于扩展
3. **类型安全**: 全量类型提示，减少运行时错误
4. **状态机**: 清晰的状态转换，避免行为冲突

### 游戏设计
1. **中式文化**: 武术流派、中文界面、传统配色
2. **角色差异化**: 力量/速度/防御三种定位，各有优劣
3. **深度战斗**: 特殊招式+增益效果，策略空间大
4. **即时反馈**: 连击数、血条渐变、击退效果，打击感强

### 平衡性考虑
- 少林僧: 高血高伤但速度慢，适合正面硬刚
- 峨眉剑客: 速度快范围远但血薄，适合游走风筝
- 武当道士: 防御强反伤高，适合防反流
