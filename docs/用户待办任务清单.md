4. **测试游戏**（2分钟）
   ```bash
   cd /Users/rookie/todo/python/chinese-fighter
   ```

**现状**: 游戏只有音效，无背景音乐

**执行步骤**:

1. **查看音乐资源指南**
   ```bash
   python3 tools/download_background_music.py
   ```

2. **选择并下载音乐**
   
   推荐来源: Incompetech (Kevin MacLeod)
   - 访问: https://incompetech.com/music/royalty-free/
   - 搜索: "Dances and Dames"（战斗）
   - 搜索: "Thatched Villagers"（菜单）
   - 下载MP3

3. **转换为OGG**（可选，推荐）
   ```bash
   # 如果有ffmpeg
   brew install ffmpeg  # macOS
   
   ffmpeg -i menu.mp3 -c:a libvorbis -q:a 4 assets/music/menu.ogg
   ffmpeg -i battle.mp3 -c:a libvorbis -q:a 4 assets/music/battle.ogg
   ```

- 检查: `python3 --version` (需要3.x)
- 检查: `pip3 list | grep pygame`
