## 🎭 角色设计

### 基础人形结构
```
头部: 圆形（直径 size/6）
身体: 矩形（宽 size/3, 高 size/3）
四肢: 线条（粗细3-5像素）
颜色: 3色方案（主色/强调色/高光色）
```

### 动画状态
1. **idle** (4帧) - 站立呼吸动画
2. **walk** (6帧) - 行走动画，腿部摆动
3. **jump** (4帧) - 跳跃姿势，腿部弯曲
4. **attack_light** (4帧) - 轻攻击，手臂前伸
5. **attack_heavy** (6帧) - 重攻击，蓄力挥拳
6. **hit** (3帧) - 受击后仰
