## 🐛 已修复的关键Bug

### 1. 中文字体乱码 ✅
**问题**: 游戏启动后中文显示为方块  
**原因**: pygame 默认字体不支持中文  
**修复**: 自动检测系统中文字体（宋体/微软雅黑/文泉驿）
```python
def get_chinese_font(size):
    fonts = [
        "/System/Library/Fonts/Supplemental/Songti.ttc",  # macOS
        "C:/Windows/Fonts/msyh.ttc",  # Windows
    ]
    # ... 自动检测
```

### 2. 角色无法跳跃 ✅
**问题**: 按W/↑键完全无反应  
**原因**: 输入处理顺序错误，跳跃状态被移动逻辑覆盖  
**修复**: 重新排序优先级（格挡→跳跃→移动→攻击）
```python
# 修复前
if move_input != 0:
    self.state = WALK
else:
    self.state = IDLE  # ← 覆盖了跳跃状态

if keys[jump]:
    self.state = JUMP  # ← 被上面覆盖

# 修复后
if keys[jump]:
    self.state = JUMP  # ← 优先设置
    
# 只在地面且非跳跃时才改变 IDLE/WALK
if on_ground and self.state != JUMP:
    ...
```

### 3. 重复受击Bug ✅
**问题**: 一次攻击造成多次伤害  
**原因**: 攻击框持续激活，每帧触发碰撞  
**修复**: `AttackBox.hit_targets` 集合记录已击中目标

### 4. 击退过强/"滑冰"问题 ✅
**问题**: 重攻击后角色滑行太远  
**修复**: 地面摩擦力 `FRICTION=0.85`
