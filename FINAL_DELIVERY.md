## 🎮 完整操作指南

### 基础操作
```
玩家1（红方）:
  移动: A(左) D(右) W(跳)
  攻击: F(轻) G(重)
  防御: H(格挡)
  特殊: Q(招式1) E(招式2)

玩家2（蓝方）/ AI:
  移动: ← → ↑
  攻击: J(轻) K(重)
  防御: L(格挡)
  特殊: Q(招式1) E(招式2)
```

### 快捷键
```
F1: 少林僧 vs 峨眉剑客
F2: 峨眉剑客 vs 少林僧
F3: 武当道士 vs 少林僧
F4: 武当道士 vs 峨眉剑客
F5: 启用/禁用AI模式（单人/双人切换）
F6: 切换AI难度（EASY → MEDIUM → HARD）
F7: 切换AI性格（AGGRESSIVE → DEFENSIVE → BALANCED）
R:  重新开始（游戏结束后）
ESC: 退出游戏
```
