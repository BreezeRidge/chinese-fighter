## 🐛 已修复的Bug

### v0.5.1 紧急修复 ⭐
**问题**：
- ❌ 不能跳跃
- ❌ 不能攻击
- ❌ 血条不减少

**根本原因**：
```python
# buff_effects.py 设置了混合类型
fighter.active_buffs["太极气盾"] = 4.0          # 数值（时间）
fighter.active_buffs["reflect_damage"] = 0.5   # 数值（反伤比例）

# _update_buffs() 对所有值执行减法
self.active_buffs[buff_name] -= dt  # ❌ 非数值类型会报错
```

**修复方案**：
```python
# 添加类型检查
if isinstance(buff_value, (int, float)) and buff_value > 0:
    self.active_buffs[buff_name] = buff_value - dt
```
