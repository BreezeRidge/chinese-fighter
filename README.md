# 武林争霸 (Wulin Champions)

**中式横版格斗游戏** - 基于 pygame-ce 开发

---

## 🎮 游戏简介

一款以中国武术为主题的 2D 格斗游戏，玩家可以选择不同武术流派的角色进行 1v1 对战。

**当前版本**: v0.2.0 Alpha（重构版：模块化+多角色+连击系统）

---

## 🕹️ 操作说明

### 玩家 1（红方）
- **移动**: A/D 键
- **跳跃**: W 键
- **轻攻击**: F 键（伤害 5，快速）
- **重攻击**: G 键（伤害 15，击退）
- **防御**: H 键（未实现）

### 玩家 2（蓝方）
- **移动**: ← → 方向键
- **跳跃**: ↑ 方向键
- **轻攻击**: J 键
- **重攻击**: K 键
- **防御**: L 键（未实现）

### 通用
- **ESC**: 退出游戏
- **R**: 游戏结束后重新开始

## 🚀 快速开始

### 安装依赖

## 🎯 设计理念

### 中式元素
- **武术流派**: 少林（力量）、武当（平衡）、峨眉（速度）
- **美术风格**: 水墨画风 + 传统配色（朱红、墨黑、青绿）
- **文化符号**: 中文界面、传统乐器音效

### 游戏手感
- **硬直系统**: 攻击/受击期间无法行动，确保打击感
- **击退效果**: 重攻击产生明显物理反馈
- **连击奖励**: 持续攻击提升伤害（计划中）
