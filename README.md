# 🥋 武林争霸 (Wulin Champions)

一款以中国武术为主题的 2D 格斗游戏，支持双人对战和单人AI对战。

**当前版本**: v1.3.0（特殊招式专属动画）✅  
**开发状态**: 生产就绪，完全可玩  
**项目规模**: 3082行代码（核心），4217行（含测试），11个模块

---

## ✨ 特色功能

### 🎮 游戏模式
- **双人对战**: 本地双人格斗
- **单人AI对战**: 挑战智能AI（3种难度，3种性格）⭐

### 👥 角色系统
- **4个武术流派角色**
  - 默认角色（平衡型）
  - 少林僧（力量型）- 高血高伤
  - 峨眉剑客（速度型）- 快速灵活
  - 武当道士（防御型）- 格挡反伤
- **8个独特特殊招式**（Q/E键触发）⭐
- **特殊招式专属动画系统**（v1.3.0）⭐⭐⭐
- **增益效果系统**（速度增强、减伤、无敌、反伤）

### 🎨 高质量角色精灵图系统 ⭐⭐⭐ (v1.1.0)
- **开源CC0授权精灵图**（来自OpenGameArt.org）
  - Chibi风格格斗角色
  - 512x512像素精灵表（8x8网格）
  - 每帧64x64像素，高清细节
  
- **8种流畅动画**（每种8帧）
  - Idle/站立动画：8帧循环（0.15秒/帧）
  - Walk/行走动画：8帧循环（0.08秒/帧）
  - Jump/跳跃动画：8帧循环（0.12秒/帧）
  - Attack Light/轻攻击：8帧（0.08秒/帧）
  - Attack Heavy/重攻击：8帧（0.1秒/帧）
  - Hurt/受伤动画：8帧（0.08秒/帧）
  
- **动作真实性提升**
  - 帧数提升：3-6帧 → 8帧（+33%~167%流畅度）
  - 文件质量：7-8KB → 19KB（高质量精灵图）
  - 动画速度优化：更快的攻击动画（0.08秒/帧）

### 🌈 角色差异化调色板系统 ⭐⭐⭐ (v1.2.0)
- **4个独特配色角色**（HSV色调偏移技术）
  - **默认角色**：原色（灰褐色基调）- 平衡型
  - **少林僧**：橙黄色（+30°色调）- 象征金刚之力
  - **峨眉剑客**：青蓝色（+200°色调）- 象征灵动之气
  - **武当道士**：青绿色（+120°色调）- 象征道法自然
  
- **视觉辨识度大幅提升**
  - 角色一眼可辨（不再是同一外观）
  - 颜色与门派特征匹配
  - 保持原精灵图细节和质量
  
- **技术实现**
  - 使用HSV色彩空间进行色调偏移
  - 保留黑色轮廓和白色高光
  - 可调节饱和度和明度增强视觉效果

### 🎬 特殊招式专属动画系统 ⭐⭐⭐ (v1.3.0)
- **8种完整动画**（充分利用精灵图资源）
  - Idle/站立：8帧（行0）
  - Walk/行走：8帧（行1）
  - **Dash/冲刺**：8帧（行2）⭐ 新增
  - Jump/跳跃：8帧（行3）
  - Attack Light/轻攻击：8帧（行4）
  - Attack Heavy/重攻击：8帧（行5）
  - Hit/受伤：8帧（行6）
  - **Charge/蓄力**：8帧（行7）⭐ 新增

- **3种招式动画类型**
  - **Dash冲刺类**：快速移动招式（0.06秒/帧）
    - 峨眉"飞燕斩"：前冲斩击
    - 峨眉"疾风步"：速度增益
    - 默认"闪避"：无敌闪避
  - **Charge蓄力类**：力量爆发招式（0.12秒/帧）
    - 少林"罗汉拳"：蓄力重击
    - 少林"金刚身"：防御增益
    - 武当"太极推手"：击退推手
    - 武当"太极气盾"：反伤护盾
  - **Attack攻击类**：普通强化攻击
    - 默认"普通重击"：增强攻击

- **视觉表现力提升**
  - 招式类型一目了然（冲刺/蓄力/攻击）
  - 门派特色更加鲜明
  - 战斗节奏感更强

### 🎨 视觉效果
- **水墨画风格场景**（竹林/武馆）
- **粒子特效系统**
  - 普通攻击：橙黄粒子
  - 重攻击：红橙粒子爆发
  - 特殊招式：金色粒子
  - 格挡：亮黄火花
- **屏幕震动效果**
- **高质量精灵图动画**（6种动画×8帧）⭐⭐⭐

### 🔊 音效反馈
- **7种程序生成音效**
  - 轻/重/特殊攻击音效
  - 格挡音效
  - 跳跃音效
  - 增益激活音效
  - KO音效

### 🤖 战术AI系统 ⭐⭐⭐ (v1.0.0)
- **3种难度**：EASY / MEDIUM / HARD
  - 反应时间：0.5s / 0.3s / 0.15s
  - 实测接近速度：328帧 / 252帧 / 170帧（难度正确分层）
- **3种性格**：AGGRESSIVE / DEFENSIVE / BALANCED
  - 进攻型：70%重攻击，高压迫
  - 防守型：80%轻攻击，快速试探
  - 平衡型：50/50攻击分布
- **战术决策系统**
  - **命中确认追击**：检测对手掉血→推进连段
  - **硬直窗口惩罚**：对手受击状态时立即压迫
  - **血量感知策略**：落后时激进反击，领先时稳健防守
  - **精确距离判断**：基于角色真实攻击范围（light_range/heavy_range）
- **连段系统**
  - 轻攻击起手→命中确认→第2-3段重攻击收尾
  - AI能对静止对手打出40次命中直至KO
- **智能招式管理**
  - 只在招式冷却完毕时释放特殊招式（避免空按浪费决策帧）


## 🎮 操作指南

### 基础操作

**玩家1（红方）**
```
移动:   A(左) D(右) W(跳跃)
攻击:   F(轻攻击) G(重攻击)
防御:   H(格挡)
特殊:   Q(特殊招式1) E(特殊招式2)
```

**玩家2（蓝方）或 AI**
```
移动:   ← → ↑
攻击:   J(轻攻击) K(重攻击)
防御:   L(格挡)
特殊:   Q(特殊招式1) E(特殊招式2)
```

### 快捷键

```
F1: 少林僧 vs 峨眉剑客
F2: 峨眉剑客 vs 少林僧
F3: 武当道士 vs 少林僧
F4: 武当道士 vs 峨眉剑客
F5: 启用/禁用AI模式（单人/双人切换）⭐
F6: 切换AI难度（EASY → MEDIUM → HARD）⭐
F7: 切换AI性格（AGGRESSIVE → DEFENSIVE → BALANCED）⭐
R:  重新开始（游戏结束后）
ESC: 退出游戏
```

## 🤖 AI使用指南

### 启用AI模式
1. 启动游戏
2. 按 **F5** 键启用AI模式
3. 玩家1 vs AI对手（玩家2由AI控制）

### 调整AI难度
- 按 **F6** 循环切换难度
  - **EASY**: 反应慢，攻击频率30%，适合新手
  - **MEDIUM**: 正常反应，攻击频率50%，适合练习
  - **HARD**: 快速反应，攻击频率70%，具有挑战性

### 调整AI性格
- 按 **F7** 循环切换性格
  - **AGGRESSIVE**: 进攻型，70%使用重攻击
  - **DEFENSIVE**: 防守型，80%使用轻攻击
  - **BALANCED**: 平衡型，攻防兼备
