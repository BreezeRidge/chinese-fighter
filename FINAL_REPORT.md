## 🎮 完整操作指南

### 基础操作
```
玩家1: WASD移动 + F轻攻 + G重攻 + H格挡 + QE特殊招式
玩家2: 方向键 + J轻攻 + K重攻 + L格挡 + QE特殊招式
```

### 快捷键
```
F1-F4: 切换角色组合
F5:    启用/禁用AI模式 ⭐
F6:    切换AI难度 ⭐
F7:    切换AI性格 ⭐
R:     重新开始
ESC:   退出
```
