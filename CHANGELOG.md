# 武林争霸 - 变更日志

## v1.3.0 (2026-07-13) 🎬 特殊招式专属动画版本

### 重大改进 - 特殊招式动画系统 ⭐⭐⭐

#### 问题背景
- v1.2.0虽然实现了角色差异化，但特殊招式动画表现力不足
- 所有特殊招式复用重攻击动画（attack_heavy）
- 无法通过动画区分招式类型（冲刺/蓄力/攻击）
- 精灵图第2行（奔跑）和第7行（死亡）资源未充分利用

#### 解决方案：招式动画类型系统

**1. 新增动画资源**
- **Dash冲刺动画**（精灵图第2行）
  - 8帧快速奔跑动画
  - 帧速度：0.06秒/帧（更快）
  - 用途：快速移动类招式

- **Charge蓄力动画**（精灵图第7行）
  - 8帧爆气/蓄力动画
  - 帧速度：0.12秒/帧（更有力）
  - 用途：力量爆发类招式

**2. 招式分类系统**

为每个特殊招式添加 `animation_type` 属性：

| 招式名称 | 角色 | 动画类型 | 视觉效果 |
|---------|------|---------|---------|
| 飞燕斩 | 峨眉 | dash | 快速冲刺斩击 |
| 疾风步 | 峨眉 | dash | 加速奔跑 |
| 闪避 | 默认 | dash | 快速闪避 |
| 罗汉拳 | 少林 | charge | 蓄力重击 |
| 金刚身 | 少林 | charge | 爆气防御 |
| 太极推手 | 武当 | charge | 蓄力推手 |
| 太极气盾 | 武当 | charge | 护盾展开 |
| 普通重击 | 默认 | attack | 普通攻击 |

**3. 动画映射逻辑**

修改 `fighter.py` 中的 `_update_sprite_animation()` 方法：

```python
if self.state == FighterState.SPECIAL and self.active_special:
    # 根据招式类型选择动画
    animation_map = {
        "dash": "dash",          # 冲刺 → 第2行
        "charge": "charge",      # 蓄力 → 第7行
        "attack": "attack_heavy" # 攻击 → 第5行
    }
    animation_name = animation_map.get(
        self.active_special.animation_type,
        "attack_heavy"
    )
```

#### 技术实现

**修改文件**：
1. `src/special_moves.py`：
   - 为 `SpecialMove` 数据类添加 `animation_type` 字段
   - 为所有8个特殊招式指定动画类型

2. `src/sprite_system.py`：
   - 新增 `"dash"` 动画定义（行2，8帧，0.06秒/帧）
   - 新增 `"charge"` 动画定义（行7，8帧，0.12秒/帧）

3. `src/fighter.py`：
   - 优化 `_update_sprite_animation()` 方法
   - 优先检查特殊招式动画类型
   - 增益状态（BUFF）使用蓄力动画

#### 动画速度对比

| 动画类型 | 帧速度 | 总时长（8帧） | 适用场景 |
|---------|--------|-------------|---------|
| **Dash冲刺** | 0.06秒/帧 | 0.48秒 | 快速移动招式 |
| Attack Light | 0.08秒/帧 | 0.64秒 | 轻攻击 |
| Attack Heavy | 0.1秒/帧 | 0.8秒 | 重攻击 |
| **Charge蓄力** | 0.12秒/帧 | 0.96秒 | 力量爆发招式 |

- **冲刺最快**：0.48秒播完，强调速度
- **蓄力最慢**：0.96秒播完，强调力量
