# 武林争霸 - 变更日志

## v0.6.0 (2026-07-09)

### 新增功能 ⭐
- **精灵图动画系统** (`sprite_system.py`, 267行)
  - AnimatedSprite 动画精灵类
  - 6种动画（idle/walk/jump/attack_light/attack_heavy/hit）
  - 帧动画播放系统
  - 占位符自动生成
  - 水平翻转支持

- **AI对战系统** (`ai_system.py`, 315行)
  - AIController AI控制器
  - 3种难度：EASY / MEDIUM / HARD
  - 3种性格：AGGRESSIVE / DEFENSIVE / BALANCED
  - 智能决策：接近/攻击/格挡/闪避
  - 特殊招式使用

- **单人模式**
  - F5: 启用/禁用AI模式
  - F6: 切换AI难度
  - F7: 切换AI性格

### 改进
- 集成精灵图到Fighter类
- 集成AI到游戏主循环
- 更新UI快捷键提示

### 测试
- 新增 `test_ai_system.py`（5项测试，全部通过）
